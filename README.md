# TypeScript Project

A new TypeScript project scaffolded with TypeScript compiler and build tools.

## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

Install dependencies:
```bash
npm install
```

### Development

Build the project:
```bash
npm run build
```

Watch mode for development:
```bash
npm run dev
```

Run the compiled code:
```bash
npm start
```

### Project Structure

```
├── src/                 # TypeScript source files
├── dist/                # Compiled JavaScript output
├── package.json         # Project configuration
├── tsconfig.json        # TypeScript configuration
└── README.md            # This file
```

## Configuration

- **TypeScript Target**: ES2020
- **Module System**: CommonJS
- **Strict Mode**: Enabled
